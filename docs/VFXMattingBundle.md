# VFX Matting Data Bundle

This bundle provides a dedicated datagen schema + GT option set for indoor VFX/matting workflows.

## Config files

- Data schema: `infinigen/datagen/configs/data_schema/vfx_matting.gin`
- GT options: `infinigen/datagen/configs/gt_options/vfx_matting_gt.gin`
- Indoor scene preset: `infinigen_examples/configs_indoor/vfx_studio.gin`

## Example `manage_jobs` commands

### Minimal VFX matting run (single scene)

```bash
python -m infinigen.datagen.manage_jobs \
  --output_folder outputs/vfx_matting_demo \
  --num_scenes 1 \
  --pipeline_configs local_256GB.gin vfx_matting.gin vfx_matting_gt.gin indoor_background_configs.gin \
  --configs vfx_studio.gin \
  --pipeline_overrides get_cmd.driver_script='infinigen_examples.generate_indoors' \
  --overrides compose_indoors.restrict_single_supported_roomtype=True
```

### Multi-scene run with explicit frame range override

```bash
python -m infinigen.datagen.manage_jobs \
  --output_folder outputs/vfx_matting_30 \
  --num_scenes 30 \
  --pipeline_configs local_256GB.gin vfx_matting.gin vfx_matting_gt.gin indoor_background_configs.gin \
  --configs vfx_studio.gin \
  --pipeline_overrides get_cmd.driver_script='infinigen_examples.generate_indoors' iterate_scene_tasks.frame_range=[1,120] \
  --overrides compose_indoors.restrict_single_supported_roomtype=True
```

> Note: `vfx_matting.gin` is loaded from `data_schema`, while `vfx_matting_gt.gin` is loaded from `gt_options`.

## Lightweight loader example (FG/BG/alpha/mask/trimap tensors)

The snippet below shows a minimal loader pattern similar to existing dataset-loading docs. It assumes:
- `Image` is the beauty RGB frame.
- `IndexOB` is used to derive an actor mask (`> 0` as foreground, customize as needed).
- FG/BG are composed from the beauty frame and derived alpha.
- Trimap is generated by morphological erosion/dilation.

```python
from pathlib import Path
import cv2
import numpy as np
import torch


def load_vfx_matting_tensors(scene_folder: str, frame_idx: int = 1):
    scene = Path(scene_folder)

    # Example paths; adjust camera/subcam suffixes to match your run.
    rgb = cv2.imread(str(scene / f"frames/Image_0_0_{frame_idx:04d}_0.png"), cv2.IMREAD_COLOR)
    rgb = cv2.cvtColor(rgb, cv2.COLOR_BGR2RGB).astype(np.float32) / 255.0

    # IndexOB is commonly saved as EXR when GT is enabled.
    obj_index = cv2.imread(
        str(scene / f"frames/IndexOB_0_0_{frame_idx:04d}_0.exr"),
        cv2.IMREAD_UNCHANGED,
    )
    if obj_index.ndim == 3:
        obj_index = obj_index[..., 0]

    mask = (obj_index > 0).astype(np.float32)
    alpha = mask.copy()

    fg = rgb * alpha[..., None]
    bg = rgb * (1.0 - alpha[..., None])

    kernel = np.ones((7, 7), np.uint8)
    eroded = cv2.erode((mask * 255).astype(np.uint8), kernel)
    dilated = cv2.dilate((mask * 255).astype(np.uint8), kernel)
    trimap = np.full(mask.shape, 0.5, dtype=np.float32)
    trimap[eroded == 255] = 1.0
    trimap[dilated == 0] = 0.0

    return {
        "fg": torch.from_numpy(fg).permute(2, 0, 1),
        "bg": torch.from_numpy(bg).permute(2, 0, 1),
        "alpha": torch.from_numpy(alpha).unsqueeze(0),
        "mask": torch.from_numpy(mask).unsqueeze(0),
        "trimap": torch.from_numpy(trimap).unsqueeze(0),
    }
```
